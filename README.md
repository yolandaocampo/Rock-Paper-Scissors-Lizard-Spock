🖖 3. Rock, Paper, Scissors, Lizard, Spock
🇺🇸 English

Description:
An expanded version of the classic "Rock, Paper, Scissors" game, featuring two additional elements: Lizard and Spock. This project was developed to practice complex conditional structures and enumerations (enums) in C#, providing an intuitive interface where the player competes against the computer's AI.

Game Rules:

Scissors cuts Paper; Paper covers Rock.

Rock crushes Lizard; Lizard poisons Spock.

Spock smashes Scissors; Scissors decapitates Lizard.

Lizard eats Paper; Paper disproves Spock.

Spock vaporizes Rock; Rock crushes Scissors.

Features:

Computer Selection: Random choice logic for a fair challenge.

Win Logic: A robust condition-checking system to determine the winner, a tie, or a loss.

GUI (Graphical User Interface): Interactive buttons representing each choice and immediate result feedback.

Score Tracker: A system to keep track of user wins versus computer wins.

Technologies:

C# | .NET Framework | WinForms

🇧🇷 Português

Descrição:
Uma versão expandida do clássico jogo "Pedra, Papel e Tesoura", que inclui dois novos elementos: Lagarto e Spock. O projeto foi desenvolvido para praticar estruturas condicionais complexas e enumerações em C#, oferecendo uma interface intuitiva onde o jogador enfrenta a inteligência do computador.

Regras do Jogo:

Tesoura corta Papel; Papel cobre Pedra.

Pedra esmaga Lagarto; Lagarto envenena Spock.

Spock derrete Tesoura; Tesoura decapita Lagarto.

Lagarto come Papel; Papel contesta Spock.

Spock vaporiza Pedra; Pedra quebra Tesoura.

Funcionalidades:

Seleção do Computador: Escolha aleatória baseada em lógica de probabilidade.

Lógica de Vitória: Sistema robusto de verificação de condições para determinar o vencedor, empate ou derrota.

Interface Gráfica: Botões com ícones ou texto representando cada escolha e exibição imediata do resultado.

Placar: Sistema para contabilizar vitórias do usuário vs. computador.

Tecnologias:

C# | .NET Framework | WinForms
